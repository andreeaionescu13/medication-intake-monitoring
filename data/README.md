# Dataset

The dataset consists of custom medication-intake videos collected for this research.

The original videos are not publicly distributed due to privacy considerations.

Dataset characteristics:

- Approximately 95 medication-intake videos
- RGB video format
- Multiple medication-related activities
- Annotated for object detection and action recognition

The dataset was used for:
- YOLOv8 pill detection
- MediaPipe pose analysis
- ResNet-18 action recognition
